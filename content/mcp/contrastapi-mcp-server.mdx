## Overview

41 security intelligence tools over remote MCP: CVE, KEV, CWE, EPSS, domain audit, IP threat reports, IOC enrichment, dependency scanning, MITRE ATLAS AI/ML attack knowledge, and MITRE D3FEND defensive techniques. Free tier: 100 credits/hour, no API key required. Streamable-HTTP transport.

## Usage

claude mcp add --transport http contrastapi https://api.contrastcyber.com/mcp/
claude mcp list

## Auth Requirements

None required for free tier (100 credits/hour per IP). Optional X-API-Key header for Pro tier (1000/hr). See https://api.contrastcyber.com/pricing

## Verification

1. `claude mcp list` shows contrastapi as connected
2. Ask Claude: "Look up CVE-2021-44228 with EPSS and KEV data"
3. Response includes verdict.completeness=full with real CVSS + EPSS percentile (~94%) + KEV listing date
