description: >-
  Automatically generates or updates API documentation when endpoint files are
  modified, supporting OpenAPI 3.1.0, Swagger 2.0, and AsyncAPI 2.0
  specifications. This hook automatically detects changes to API route files
  (JavaScript, TypeScript, Python) and generates comprehensive documentation
  using swagger-jsdoc (v6+), JSDoc, or FastAPI's built-in OpenAPI generation.
  Features include real-time documentation synchronization, multi-format output
  (JSON, YAML, Markdown), automatic endpoint discovery, parameter and response
  type extraction, and integration with Swagger UI and ReDoc for interactive
  documentation. The hook supports both annotation-based documentation (JSDoc
  comments) and automatic schema inference (FastAPI), ensuring API documentation
  stays synchronized with code changes throughout the development lifecycle.
