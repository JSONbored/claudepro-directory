Features include real-time documentation synchronization, multi-format output (JSON, YAML, Markdown), automatic endpoint discovery, parameter and response type extraction, and integration with Swagger UI and ReDoc for interactive documentation. The hook supports both annotation-based documentation (JSDoc comments) and automatic schema inference (FastAPI), ensuring API documentation stays synchronized with code changes throughout the development lifecycle."
