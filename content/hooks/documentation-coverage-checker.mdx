  coverage metrics and reporting. Features include automatic detection of
  undocumented functions and classes, multi-language documentation validation
  (JavaScript, TypeScript, Python, Go, Rust), API endpoint documentation
  completeness checking, documentation coverage metrics and reporting with
  threshold enforcement, README and changelog freshness validation, and support
  for multiple languages and documentation formats.
