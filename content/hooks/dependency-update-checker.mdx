  security analysis. This PostToolUse hook triggers when dependency manifest
  files (package.json, requirements.txt, Gemfile, go.mod, Cargo.toml,
  pyproject.toml) are modified, providing real-time dependency health
  monitoring. Supports npm-check-updates (ncu), npm outdated, pip-audit 2.9.0+,
  safety, cargo-outdated, cargo-audit, govulncheck, and bundler-audit with
  categorized update recommendations (critical, major, minor, patch), breaking
  change warnings for major version updates, security vulnerability detection
  integrated with dependency updates, multi-language support (Node.js, Python,
  Ruby, Go, Rust), and detailed update strategy guidance with changelog
  references.
