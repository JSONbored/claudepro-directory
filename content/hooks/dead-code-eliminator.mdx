description: >-
  Automatically detects and removes unused code, imports, and dependencies with
  safe deletion verification and rollback support using Knip 5.x, ts-prune,
  depcheck, autoflake, Vulture, and ESLint 9.37.0. This SessionEnd hook runs
  when Claude Code session ends and automatically analyzes the codebase for
  unused imports, dead code paths, unreferenced functions and variables,
  orphaned test files, unused dependencies, and zero-coverage files. Features
  include unused import detection and removal across multiple languages
  (JavaScript/TypeScript, Python, and more), dead code path identification using
  static analysis and AST parsing, unreferenced function and variable detection
  with confidence scoring, orphaned test file cleanup with verification and
  safety checks, safe deletion with backup and rollback capabilities creating
  timestamped backups before removal, bundle size impact analysis after cleanup
  showing size reduction metrics, coverage-based dead code detection identifying
  zero-coverage files as potential dead code, and multi-language support for
  JavaScript/TypeScript (Knip 5.x, ts-prune, ESLint 9.37.0), Python (autoflake,
  Vulture), and dependency analysis (depcheck).
