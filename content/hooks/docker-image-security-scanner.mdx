  vulnerability reporting, base image vulnerability detection and
  recommendations with alternative base image suggestions, malware and rootkit
  scanning in container images using Trivy and Grype, security best practices
  validation (non-root user, minimal layers, version pinning, secrets
  detection), and integration with Trivy 0.52.0+, Grype, and Docker Scout for
  comprehensive vulnerability coverage.
