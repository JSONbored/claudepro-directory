  This PostToolUse hook provides comprehensive GraphQL schema validation with
  comprehensive GraphQL schema syntax validation using graphql-inspector
  (recommended) or basic pattern matching with fallback methods, breaking change
  detection with detailed impact analysis comparing current schema with previous
  versions and identifying type removals, field modifications, and enum changes,
  schema evolution tracking and version comparison with timestamped backups and
  automatic backup management, multi-tool validation support (graphql-inspector,
  graphql-schema-linter) with automatic tool installation and graceful fallback,
  custom directive and scalar type validation with federation directive checking
  (@key, @external, @provides, @requires), schema complexity analysis and
  performance warnings with nesting depth analysis and circular reference
  detection, federation schema compatibility checking with Apollo Federation
  support and schema structure validation, and auto-backup and rollback
  capabilities for schema changes with timestamped backups and automatic cleanup
  of old backups. The hook automatically detects GraphQL schema files (.graphql,
  .gql, *schema*), validates syntax and structure, detects breaking changes,
  analyzes schema quality, checks federation compatibility, and provides
  comprehensive validation reports with actionable recommendations.
