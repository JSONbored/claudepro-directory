# Configure Claude as AWS Cloud Architect

*Complete guide updated for September 2025*

Enable Claude to provide expert AWS architecture advice and best practices. This tutorial shows you how to configure **Aws Cloud Architect** to transform Claude into a cloud architecture planning.

## Prerequisites

- Claude Desktop (latest version) OR Claude Pro web access
- AWS account
- Basic cloud knowledge

## Quick Setup (2 minutes)

### Method 1: Claude Desktop

#### 1. Locate Configuration File

**macOS:**
```bash
~/Library/Application Support/Claude/claude_desktop_config.json
```

**Windows:**
```bash
%APPDATA%\Claude\claude_desktop_config.json
```

#### 2. Add Rule Configuration

```json
{
  "rules": [
    {
      "name": "Aws Cloud Architect",
      "description": "Expert AWS architect with deep knowledge of cloud services, best practices, and Well-Architected Framework",
      "active": true,
      "content": "// Copy the complete rule from claudepro.directory/rules/aws-cloud-architect"
    }
  ]
}
```

#### 3. Restart Claude Desktop

Completely restart Claude Desktop for changes to take effect.

### Method 2: Claude Pro Web

#### 1. Start New Conversation

Create a fresh conversation for best results.

#### 2. Load System Prompt

Copy the system prompt from [our rule page](/rules/aws-cloud-architect) and paste at conversation start.

#### 3. Verify Setup

Type: "What is your expertise area?"
