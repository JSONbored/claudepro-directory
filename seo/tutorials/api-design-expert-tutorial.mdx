# How to Configure Claude as an API Design Expert

*Complete guide updated for September 2025*

Transform Claude into a comprehensive API design specialist with expert knowledge of RESTful APIs, GraphQL, and OpenAPI. This tutorial shows you how to configure **Api Design Expert** to transform Claude into a API development and design.

## Prerequisites

- Claude Desktop (latest version) OR Claude Pro web access
- Basic understanding of APIs
- Claude Desktop or Pro access

## Quick Setup (2 minutes)

### Method 1: Claude Desktop

#### 1. Locate Configuration File

**macOS:**
```bash
~/Library/Application Support/Claude/claude_desktop_config.json
```

**Windows:**
```bash
%APPDATA%\Claude\claude_desktop_config.json
```

#### 2. Add Rule Configuration

```json
{
  "rules": [
    {
      "name": "Api Design Expert",
      "description": "Transform Claude into a comprehensive API design specialist focused on RESTful APIs, GraphQL, OpenAPI, and modern API architecture patterns",
      "active": true,
      "content": "// Copy the complete rule from claudepro.directory/rules/api-design-expert"
    }
  ]
}
```

#### 3. Restart Claude Desktop

Completely restart Claude Desktop for changes to take effect.

### Method 2: Claude Pro Web

#### 1. Start New Conversation

Create a fresh conversation for best results.

#### 2. Load System Prompt

Copy the system prompt from [our rule page](/rules/api-design-expert) and paste at conversation start.

#### 3. Verify Setup

Type: "What is your expertise area?"
