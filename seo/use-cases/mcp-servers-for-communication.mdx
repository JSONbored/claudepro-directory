## Implementation Guide

### Step 1: Choose Your MCP Server

Based on your specific needs for team communication, we recommend starting with [Discord Integration MCP](/mcp/discord-integration) for community management or [Slack Integration MCP](/mcp/slack-integration) for enterprise teams.

### Step 2: Installation

Add your chosen MCP server to your Claude configuration file:

**Configuration file location:**  
`~/Library/Application Support/Claude/claude_desktop_config.json` (macOS)  
`%APPDATA%/Claude/claude_desktop_config.json` (Windows)

### Step 3: Configuration

Add the appropriate configuration to your `claude_desktop_config.json` file:

**For Discord Integration:**
```json
{
  "mcpServers": {
    "discord-integration": {
      "command": "npx",
      "args": ["@discord/mcp-server"]
    }
  }
}
```

**For Slack Integration:**
```json
{
  "mcpServers": {
    "slack-integration": {
      "command": "npx",
      "args": ["@slack/mcp-server"]
    }
  }
}
```
